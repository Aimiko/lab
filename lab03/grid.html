การใช้งาน Grid

Post by Chat-usa sukkasem, feb 9, 2017.
Grid Form

Grid system ของ Bootstrap แบ่งออกเป็น 12 คอลัมน์ ถ้าให้เห็นภาพง่ายขึ้นก็น่าจะต้องบอกว่า เหมือนกับตาราง 1 แถว มี 12 คอลัมน์ประมาณนั้น โดยคอลัมน์ของ Grid system แบ่งออกเป็น 4 ประเภทคือ ดังภาพด้านล่าง


สรุปจากภาพด้านบนง่ายๆดังนี้
XS : ใช้กับหน้าจอขนาดเล็กกว่า Ipad แนวตั้งลงมา (หน้าจอสมาร์ทโฟนต่างๆ)
SM : ขนาดหน้าจอตั้งแต่ Ipad แนวตั้ง
MD : ขนาดความกว้างต่ำสุดที่หน้าจอ Ipad แนวนอน
LG : หน้าจอที่กว้างกว่า 1200 pixels ขึ้นไป หรือให้เหมาะสมกับขนาดหน้าจอโน้ตบุคมาตราฐาน
